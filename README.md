# avito_tests

Для запуска тестов выполнить:

### issue-01

```python -m doctest -v -o NORMALIZE_WHITESPACE test_1.py```
 
или Run -> 'Doctest encode'

### issue-02

```pytest test_2.py``` или ```python -m pytest test_2.py```

### issue-03

```python -m unittest test_3.py```


### issue-04
